This is a web applciation where the backend is written in NodeJS using MongoDb as the primary database. The frontend is created using ReactJS and is connected to the backend via HTTP. 

Steps to run the backend
1) Open Backend in Terminal
2) Type yarn install
3) Type yarn start

Steps to run the frontend
1) Open Backend in Terminal
2) Type yarn install
3) Type yarn start



The user needs to be logged in to access the pages. There are mainly 5 pages 
1) Home 
2) About
3) Contact
4) Jobs
5)Login

Created a Dynamic listing using react map with data retrived from DB on job page
